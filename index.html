Battleship

Based on Battleship example in "Heads First JavaScript Programming" chapter 8

Modifications include:

  • Use array length instead of constants on ships array and ship.location array. This allows any number of ships and different ships length by merely modifying the initial values in the ships array
  • Moved images and style information to appropriate subdirectory
  • Added name property to ship object to display when ship is sunk.
  • Replaced literal construction of a 'ship' with a 'Ship' construction function

Current version has a single 4 ship (Battleship), two 3 ships (Cruiser), one 2 ship (Destroyer).
